# 空空如也GameHub 游戏信息检索网站需求文档

## 1. 项目定位

空空如也GameHub 是一个游戏信息检索网站，面向玩家提供游戏检索、分类浏览、游戏列表、独立详情页和下载地址展示能力。

当前版本已移除账户注册和论坛模块，首页只聚焦游戏信息展示。广告位能力在后端保留为预留接口，当前前端不展示广告布局。

## 2. 当前功能范围

### 2.1 游戏检索

- 用户可以在首页顶部搜索框输入关键词。
- 支持按游戏名称、简介、标签、工作室检索。
- 检索结果通过 `/api/games` 返回。
- 检索结果支持分页。

### 2.2 游戏分类

- 首页展示游戏类型分类。
- 点击分类后展示该类型下的游戏。
- 一个游戏可以对应多个类型。
- 多类型筛选时，只要任意一个类型命中即可返回该游戏。

### 2.3 游戏列表

首页游戏列表展示：

- 游戏封面。
- 游戏名称。
- 游戏所有类型。
- 发行年份，缺失时不展示。
- 底部胶囊：首个类型、游戏大小；游戏大小缺失时不展示。
- 查看详情按钮。

### 2.4 焦点游戏位

首页顶部保留焦点游戏位，用于展示当前分页结果中的第一款游戏。

焦点游戏位展示：

- 封面。
- 类型。
- 游戏名称。
- 查看详情按钮。

焦点游戏位不展示简介摘要、评分、年份和平台。

### 2.5 游戏详情

点击“查看详情”进入独立详情页。

详情页展示：

- 游戏封面。
- 游戏类型。
- 游戏名称。
- 仅展示下载地址，地址不可点击、不可被鼠标选中复制。
- 评分。
- 发行年份。
- 游戏大小。
- 支持平台。
- 下载地址按钮。
- 详情主体上方需要提供轻量清晰的面包屑返回入口。

### 2.6 分页

首页需要支持分页浏览全部游戏数据。

分页位置：

- 游戏库工具栏右侧提供顶部分页。
- 游戏列表底部保留底部分页。

### 2.7 广告位预留

当前前端不展示广告位。

后端保留广告位接口：

```text
GET /api/ads?placement=
```

当前保留广告位：

- `home_top`
- `sidebar`

## 3. 已移除功能

以下模块已从当前版本移除：

- 账户注册。
- 用户登录。
- 登录会话。
- 权限校验。
- 论坛帖子列表。
- 论坛发帖。
- 论坛评论。
- 论坛信息流广告位。

后续如需恢复，应作为新需求重新设计接口、页面、数据库表和权限规则。

## 4. 用户角色

当前版本不区分登录用户和游客，所有访问者均可：

- 浏览游戏列表。
- 搜索游戏。
- 使用分类筛选。
- 查看游戏详情。
- 打开下载地址。

## 5. 页面范围

| 页面 | 文件 | 功能 |
| --- | --- | --- |
| 首页/游戏库 | `frontend/preview.html` | 搜索、分类、焦点位、游戏列表、分页 |
| 详情页 | `frontend/detail.html` | 游戏详情、下载入口、返回游戏库 |

已删除页面：

| 页面 | 原文件 | 状态 |
| --- | --- | --- |
| 账户页 | `frontend/auth.html` | 已删除 |
| 论坛页 | `frontend/forum.html` | 已删除 |

## 6. 接口范围

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| `/api/health` | GET | 健康检查 |
| `/api/categories` | GET | 获取游戏类型 |
| `/api/games` | GET | 游戏列表、搜索、分类筛选、分页 |
| `/api/games/{slug}` | GET | 游戏详情 |
| `/api/ads` | GET | 广告位预留接口 |

已移除接口：

- `/api/auth/register`
- `/api/auth/login`
- `/api/me`
- `/api/forum/posts`
- `/api/forum/posts/{id}`
- `/api/forum/posts/{id}/comments`

## 7. 数据要求

当前核心数据来自 `data/fufu_quark_only.csv` 导入结果。

游戏数据要求：

- `games` 保存游戏主信息。
- `categories` 保存去重后的游戏类型。
- `game_categories` 保存游戏和类型的多对多关系。
- `games.download_url` 保存游戏下载地址。
- 游戏大小从简介或详情中的 `资源大小：` 提取并在前端展示。

## 8. 验收标准

- 首页可以直接展示数据库导入后的游戏列表。
- 搜索功能可以返回匹配游戏。
- 分类筛选使用游戏多类型关系，任意类型命中即可返回。
- 分页可以访问全部游戏。
- 点击查看详情跳转到独立详情页。
- 游戏详情页可以展示下载地址。
- 首页不展示账户注册入口。
- 首页不展示论坛入口。
- 访问已移除页面应返回不可用或找不到页面。
- 后端不再创建账户、会话、论坛帖子、论坛评论表。
