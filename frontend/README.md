# 空空如也GameHub Frontend

空空如也GameHub 前端用于游戏信息检索、分类筛选、游戏列表、分页和详情页展示。

当前静态运行入口：

- `preview.html`
- `detail.html`

账户注册和论坛页面已删除。

当前静态页设计基准：

- 首页和详情页顶部不展示“游戏库”导航按钮。
- 详情页正常状态不展示“返回游戏库”和面包屑导航。
- 详情页顶部使用品牌栏和装饰轨。
- 首页 H1-1 焦点位展示封面、标题、简介摘要和查看详情按钮。
- 首页 H4 普通游戏卡片只展示封面、游戏名、类型信息行和查看详情按钮。
- 首页 H4 普通游戏卡片不展示年份和游戏大小。
- 默认封面和 CSS 已使用资源版本号降低缓存影响。

## 本地访问

推荐通过后端统一托管访问：

```text
http://127.0.0.1:8000/
```

## React 工程

`frontend/src/` 保留 React + Vite + TypeScript 工程源码，用于后续正式前端工程化开发。

如需运行 React 开发服务：

```bash
cd frontend
npm install
npm run dev
```
