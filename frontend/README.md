# 空空如也GameHub Frontend

空空如也GameHub 前端用于游戏信息检索、分类筛选、游戏列表、分页和详情页展示。

当前静态运行入口：

- `preview.html`
- `detail.html`

账户注册和论坛页面已删除。

## 本地访问

推荐通过后端统一托管访问：

```text
http://127.0.0.1:8000/
```

## React 工程

`frontend/src/` 保留 React + Vite + TypeScript 工程源码，用于后续正式前端工程化开发。

如需运行 React 开发服务：

```bash
cd frontend
npm install
npm run dev
```
